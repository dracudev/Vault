#### Default Font

```css
body {
  font-family: Arial, sans-serif;
}
```

#### Font Style and Size

```css
h1 {
  font-size: 24px;
  font-style: italic;
  font-weight: bold;
}
```

#### Text Transformations

```css
p {
  text-transform: uppercase; /* Changes text to uppercase */
  text-align: center; /* Centers the text */
  letter-spacing: 2px; /* Adjusts spacing between letters */
}
```
