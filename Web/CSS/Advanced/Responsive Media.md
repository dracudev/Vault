#### Media Queries

Change styles based on screen size.

```css
@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}
```

#### Relative Sizes

Use units like `em`, `rem`, `%` for flexible designs.

```css
p {
  font-size: 2rem; /* Scales based on the root of the document */
}
```
