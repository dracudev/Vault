### Arithmetic Operators

- **Addition**: `+`
```c
int sum = a + b;
```
- **Subtraction**: `-`
```c
int difference = a - b;
```
- **Multiplication**: `*`
```c
int product = a * b;
```
- **Division**: `/`
```c
int quotient = a / b;
```
- **Modulus**: `%`
```c
int remainder = a % b;
```
### Relational Operators

- **Equal to**: `==`
```c
if (a == b) { /* ... */ }
```
- **Not equal to**: `!=`
```c
if (a != b) { /* ... */ }
```
- **Greater than**: `>`
```c
if (a > b) { /* ... */ }
```
- **Less than**: `<`
```c
if (a < b) { /* ... */ }
```
- **Greater than or equal to**: `>=`
```c
if (a >= b) { /* ... */ }
```
- **Less than or equal to**: `<=`
```c
if (a <= b) { /* ... */ }
```
